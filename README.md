# English 🪟 Janala

A beginner-friendly English vocabulary learning web app. Browse lessons, search words, hear pronunciations, and save your favorites — all in one place.

## 🌟 Features

- **Lesson-based vocabulary** — Words organized by levels, loaded from a live API
- **Word details** — View meaning, pronunciation, example sentence, and synonyms in a modal
- **Text-to-speech** — Hear the correct pronunciation of any word with one click
- **Search** — Find any word instantly across the entire vocabulary library
- **Save words** — Bookmark favorite words with a heart icon and review them anytime
- **FAQ section** — Collapsible frequently asked questions
- **Responsive design** — Works on mobile, tablet, and desktop

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (Vanilla)
- [Tailwind CSS v4](https://tailwindcss.com/) + [DaisyUI v5](https://daisyui.com/)
- [Font Awesome 7](https://fontawesome.com/)
